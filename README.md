## ü§º‚Äç‚ôÇÔ∏è How to

```bash
$ export VPN_NAME=gateway   # your server name, any way you like
$ export VPN_HOST=127.0.0.1 # your server ip

$ cat ansible/hosts.tpl.ini \
  | sed "s/{{.Name}}/${VPN_NAME}/g" \
  | sed "s/{{.Host}}/${VPN_HOST}/g" \
  > ansible/hosts

$ make
```
